Verifica se os itens de um Array existem em outro Array. Se ao menos um item do primeiro Array existir no segundo Array a função retorna true. Deve ser notado que a verificação é sempre feita se somente itens de array_a existem em array_b e nunca o oposto. Deve-se notar também que a verificação considera um match apenas a primeira ocorrência do item correspondente em array_b. Ou seja, se array_a possui um valor x e arry_b possui dois valores x, a primeira ocorrência é quem dará retorno.

Nota: O modo como esta função trabalha e retorna um resultado pode ser configurado com o uso dos argumentos de controle.

Parâmetros:

  • array_a: array

    • Recebe um Array para ser analisado. Será verificado se ao menos um valor corresponde a algum valor existente em array_b.

  • array_b: array

    • Recebe um Array para se comparado com um valor de array_a.

  • control_arg: string | number

    • Pode receber argumentos de controle do tipo string e/ou um número inteiro. São eles:

      • "all":

        A função só retorna true se todos os itens de array_a existirem em array_b, caso contrário o retorno é false.

      • "list":

        Retorna um Array contendo um objeto para cada item de array_a que corresponder. Cada objeto contém o item correspondente, o index do item correspondente em array_a, o index do item correspondente em array_b e o tipo do item.
